# Changelog

All notable changes to the Obsidian Memos Sync Plugin will be documented in this file.

## [1.0.1] - 2024-12-10

### Fixed
- 修复了与 Memos API 兼容性的问题
- 更新了 API 请求格式以匹配 Memos v0.16.3
- 改进了错误处理和日志输出

### Added
- 添加了版本兼容性说明
- 添加了更详细的错误信息提示

### Changed
- 优化了 API 请求逻辑
- 更新了文档，添加了版本要求说明

## [1.0.0] - 2024-12-04

### Initial Release
- 基本的 Memos 同步功能
- 支持手动和自动同步
- 文件按年月组织
- 支持图片和附件同步
- Markdown 内容优化
- 标签转换功能
